## Reading

On the same 3090, same command, Qwen2.5-7B is a touch faster than Llama-3.1-8B
(+8% prefill, +4% decode) and a touch leaner — explained simply by its smaller
parameter count (7.62B vs 8.03B). No surprise, and that's the point: the boring
library lets you read "what runs how fast here" across families at a glance.

**Why PPL is not cross-model comparable:** perplexity is computed over each model's
*own tokenizer and vocabulary*. Qwen and Llama tokenize wikitext differently, so
8.02 vs 7.50 says nothing about which model is "better" — only each model's
self-consistency on this text. PPL is comparable *within* a model across quants
(see R2/R2b), never *across* models. Reporting it as a head-to-head would be the
exact hype this lab refuses.
