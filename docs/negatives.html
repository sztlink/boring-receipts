How to read a negative receipt

  1. What tempting claim did it test?
  2. What part of the stack resisted?
  3. What remains true after the failure?
  4. What path does it close or redirect?

Current lesson

entity-hop path prompt == gated rerank v1 at N=500. Option B gated control failed fresh holdout. hand-written gated control is closed for now. KV q8/q8 does not beat f16 through 64K, and KV q8/q4 exits -1 before 32K in the patched-source long-context test. Each demotes a tempting claim to measured local evidence.
