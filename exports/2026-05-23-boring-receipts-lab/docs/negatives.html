How to read a negative receipt

  1. What tempting claim did it test?
  2. What part of the stack resisted?
  3. What remains true after the failure?
  4. What path does it close or redirect?

Current lesson

entity-hop path prompt == gated rerank v1 at N=500. That does not erase the 100-case signal. It demotes it to what it is: a small-slice historical signal, not a scaled quality claim.

Positive, negative and no-delta states use the same receipt structure and the same public weight.
