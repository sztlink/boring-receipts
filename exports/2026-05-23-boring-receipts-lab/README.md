# 2026-05-23-boring-receipts-lab

Static preservation package for Boring Receipts Lab.

Open `docs/index.html` for the public exhibit. Read `receipts/*.md`, `SUMMARY.md`, `PROCESS-INDEX.md` and `NEGATIVES.md` for the receipt archive.

Integrity files:

- `manifest.json` - file list, byte counts and SHA-256 hashes.
- `SHA256SUMS` - portable checksum list.

Boundary: this package preserves the proof trail and command shapes. It does not bundle vendor binaries, model weights or external datasets.
